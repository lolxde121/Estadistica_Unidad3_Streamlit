# Aplicación de Probabilidad y Prueba de Hipótesis
**Estudiante:** Edgar Solís Hernández  
**Materia:** Probabilidad y Esatdistica  
**Ingeniería:** Tecnologías de la Información e Innovación Digital

## Descripción
Esta aplicación interactiva desarrollada en Streamlit permite realizar visualizaciones de datos y ejecutar pruebas de hipótesis Z con integración de la API de Google Gemini para interpretación de resultados.

## Requisitos
- Python 3.12+
- Librerías: streamlit, pandas, numpy, plotly, scipy, google-generativeai

## Ejecución
1. Crear entorno virtual: `python3 -m venv .venv`
2. Activar entorno: `source .venv/bin/activate`
3. Instalar dependencias: `pip install -r requirements.txt`
4. Correr app: `streamlit run app.py`
